# 紫金港达美装饰 × 钉钉数字化管理系统 Demo

> 基于钉钉 AI 表格 + 自动化流程，为装饰行业连锁餐饮装修公司量身定制的轻量级数字化管理方案。

## 项目背景

深圳紫金港达美装饰工程有限公司，专注于：
- **餐饮连锁新店装修**（肯德基、麦当劳、瑞幸咖啡等）
- **商业综合体公共区域维修**（深业上城、一方城等）

本 Demo 展示如何基于**钉钉原生能力（零额外采购）**实现：项目全生命周期管理、财务收付款追踪、老板实时驾驶舱、CRM 客户关系管理，并通过 AI 自动化消除繁琐人工操作。

---

## 系统架构

```
钉钉 AI 表格（数据层）
    ├── 项目合同表       ← 合同签订→施工→验收→质保金
    ├── 收付款流水表     ← 应收/应付/开票/质保金
    ├── CRM 客户表       ← 市场部跟进记录
    └── 行政审批表       ← 报销/请假/采购

自动化流程（逻辑层）
    ├── 质保金到期提醒   ← 每日检查，提前 30 天预警
    ├── 月度财务简报     ← 每月 1 日自动汇总推送老板
    └── 新合同签订通知   ← 实时推送项目负责人+老板

老板驾驶舱（展示层）
    └── 实时汇总：在建项目数 / 本月收款 / 待收质保金 / 逾期项目
```

## 快速开始

### 1. 环境准备

```bash
git clone https://github.com/YOUR_USERNAME/zjg-dingding-demo.git
cd zjg-dingding-demo
pip install -r requirements.txt
cp .env.example .env
# 编辑 .env 填入钉钉应用凭证
```

### 2. 初始化钉钉 AI 表格结构

```bash
python scripts/init_tables.py
```

### 3. 启动自动化服务

```bash
# 质保金提醒（每日运行）
python src/automation/warranty_reminder.py

# 老板驾驶舱汇总（按需触发）
python src/dashboard/boss_summary.py
```

## 核心功能演示

### 项目全生命周期管理

| 阶段 | 操作 | 自动化动作 |
|------|------|-----------|
| 合同签订 | 录入 AI 表格 | 推送通知项目经理+老板 |
| 施工中 | 更新进度字段 | 无 |
| 竣工验收 | 标记"验收完成" | 触发开票提醒 |
| 质保期 | 系统自动追踪 | 到期前 30 天预警 |
| 质保金回收 | 更新收款状态 | 月报自动统计 |

⚠️ 逾期未收款：2 个项目，请跟进
